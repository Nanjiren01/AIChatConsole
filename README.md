## 功能

这是[Nanjiren01/AIChatWeb](https://github.com/Nanjiren01/AIChatWeb)项目的后台，目前包含的功能有

1. 成员管理
   1. 停启用
   2. 额度变动记录
   3. 手动添加额度
   4. 修改密码
2. 模型及api key管理
   1. 停启用key
   2. 使用次数查看
3. 管理员管理
   1. 添加管理员
   2. 停启用管理员
